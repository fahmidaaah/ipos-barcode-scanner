# iPOS - Barcode Scanner & Thermal Label Printing System

## 📱 Overview

A Flutter-based mobile application developed for retail inventory management. The application connects to a SQL Server database to retrieve product information using barcode scanning and supports Bluetooth thermal label printing.

---

## ✨ Features

- Splash Screen
- SQL Server Database Configuration
- Product Search using Barcode
- Product Details Display
- Bluetooth Thermal Label Printing
- Custom Label Format Support
- Simple and User-Friendly UI

---

## 🛠️ Tech Stack

- Flutter
- Dart
- SQL Server
- Bluetooth Thermal Printer
- Shared Preferences
